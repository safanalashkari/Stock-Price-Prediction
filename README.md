# 📈 Stock Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts future stock prices using historical market data.
It uses a **Random Forest Regressor** model trained on real stock market
data downloaded from Yahoo Finance.

## 🛠️ Technologies Used
- Python 3.9
- Pandas & NumPy (Data Processing)
- Matplotlib & Seaborn (Data Visualization)
- Scikit-Learn (Machine Learning Model)
- yFinance (Stock Data Download)
- Jupyter Notebook

## 📊 Dataset
- **Stock:** Apple Inc. (AAPL)
- **Period:** January 2015 – January 2024
- **Source:** Yahoo Finance
- **Features Used:** Closing Price

## 🔍 Project Workflow
1. Download real-time stock data using yFinance
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing & Scaling
4. Create 60-day training sequences
5. Train Random Forest model
6. Predict & visualize results

## 📈 Results
- Predicted stock prices closely follow actual prices
- Model evaluated using MAE and RMSE metrics
